# Notes on Real and Complex Analysis

A collection of LaTeX notes on topics in real and complex analysis.

## Contents

- **Stereographic Projections and Mobius Transformations** — Notes on stereographic projections and Mobius transformations, covering conformal mappings on the Riemann sphere.
- **Heine-Borel Theorem for Open Infinite Sets** — Exposition on the Heine-Borel theorem, exploring compactness, open covers, and finite subcovers in real analysis.
